В результате выполнения тестов и с помощью ручного тестирования были обнаружены следующие баги
1. Символ 'i' воспринимается сервером как символ '1'
  * Пример запроса
  ```
  {"string": "2i3"}
  ```
  * Ожидаемый результат
  ```
  {"number": 5}
  ```
  * Фактический результат
  ```
  {"number": 6}
  ```
